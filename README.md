# SAMMERS JEANS 

Sistema e-commerce para la venta de pantalones de jean para dama, hombre y niño.  
Permite al usuario navegar por categorías, ver precios, tallas y realizar compras con diferentes métodos de pago.

## Características principales
- Registro e inicio de sesión de usuarios.
- Visualización de productos por categoría y tallas.
- Carrito de compras y procesamiento de pedidos.
- Panel de administración para gestión de inventario y usuarios.
